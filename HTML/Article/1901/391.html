在新加坡工作，关于领薪水&加班费的那些事
更新日期：2019-01-22 19:31:42

以下信息全部来源于新加坡人力部）

什么时候领薪水？

雇主必须每个月至少1次，在薪金周期结束后的7天内支付雇员薪金。此外，所有加班费必须在薪金周期结束后的14 天内支付。

如何计算加班费？

至于加班费，如果雇佣合约薪金如下，就能受到雇佣法令的第四条款的保障，要求加班费索赔：

1） 基本月薪低于$4,500的劳力员工（俗称蓝领）

2） 基本月薪低于$2,500非劳工员工（俗称白领）

政府规定，加班费必须以每小时底薪的1.5倍计算。

同时，在雇佣合约下，雇员每天的工作时数不得超过8个小时或每星期44个小时（1个星期是指由星期一开始，结束于星期日的连续7天），不包括餐饮或休息时间。

因此，在计算加班费的时候，雇佣法令是以每周44小时工作时间为计算标准。

月薪劳力工人：1.5 x 加班工作时数x (12 x 每月底薪) /（52 周 x 44 小时）

每月底薪低于$2,250的非劳力工人：1.5 x 加班工作时数 x (12 x 每月底薪) /（52 周 x 44 小时） 每月底薪为$2,250

或以上的非劳力工人：1.5 x 加班工作时数 x (12 x $2250) /（52 周 x 44 小时）

（加班费计算方式示例）

并且，和公共假期允许公司调休不同，一旦雇员加班了，除了付加班费之外，任何雇主不能通过调休的方式补上加班时长。

除此之外，雇主是没有权利扣除雇员薪水的，除非以下几种情况。

在公司外部， 雇主可以接受法庭命令、其他主管当局作出的命令以及所得税司长的指示扣除雇员薪水。或者雇主代主管当局收回雇员应缴付的所得税、房产税或消费税之后，也是可以扣除雇员相应薪水的。

在公司内部考核系统里，在提供正当理由的情况下，雇主也可以扣薪金。

比如以下情况：

a) 雇员旷工。

b) 委托给雇员的货品或金钱是因为雇员的疏忽或过失而直接造成毁坏或遗失，所扣除金额不能超过每月薪金的25%，而且只能在确认毁坏或遗失是因为雇员的疏忽或过失所造成的，才能扣除薪金。

c) 雇主在雇员的要求下所提供的膳食费。

d) 雇主提供，雇员也接受的住宿或设施与服务费用。扣除的住宿或设施与服务费用不能超过一个月薪金的25%。

e) 雇员需还清的预支薪金、贷款或调整透支的薪金。在扣除预付款和贷款的情况下，扣除金额不应超过每月薪金的25%。

f) 缴交公积金。

g) 按雇员的书面请求，为退休金计划、公积金或其他计划所缴付的款项。这些计划必须 是为了雇员的利益而合法推出的，并且得到劳工总监的批准。

h) 经过雇员的书面同意，向任何注册的合作社协会付款。

i) 其他个别获得人力部部长批准的原因。

实际上，只要遵守公司相应规章制度，老板也很难找到借口来扣薪水。但不排除有些黑心的老板，这时候就需要小伙伴们拿起法律的武器保护自己权益了！

如果员工入职不足一个月，雇主也需要支付薪水给雇员。具体是按照实际工作天数占当月工作总天数的比例来计算。

要注意的是，上图中的“每月薪金总额”，不包括任何奖金、花红、补助津贴或者加班费。

“上月工作天总数”，不包括休息日（如周末）和非工作日，但包括公共假期。

而“雇员当月实际工作天数”则包括任何有薪假期（年假、病假、公共假期）。

范例：如何计算工作时间不到1个月的薪金 泰明的工作是5 天制，从星期一到星期五，每月底薪是$700，月薪总额为$800。他 已经呈交了辞职信，最后一个工作日是8 月26 日。他在8 月11 日至14 日拿了4天的 有薪年假，星期天是泰明的休息日。

（不足1个月的薪水计算范例）

所以小伙伴们现在明白了吗？如果对薪水有疑问，可以在发薪水的时候问公司讨要详细的薪水单。这是新加坡人力部鼓励的一种行为，能够确实保证自己的权益不受侵犯。
